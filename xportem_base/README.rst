Features
========

This module provides the foundation for the Xportem Procurement Suite with the following core features:

Product Categories
------------------

* Hierarchical product category structure with unlimited levels
* Auto-generation of category codes
* Complete name computation for easy navigation
* Color coding for visual organization
* Search enhancement with intelligent filtering

Product Tags
------------

* Hierarchical tag system for detailed product classification
* Alternative names support for multilingual environments
* Keywords for enhanced search capabilities
* Category association for better organization
* Intelligent search functionality for supplier matching

Certifications
--------------

* Comprehensive certification management
* Multiple certification types (Quality, Safety, Environmental, Industry-specific)
* Mandatory/optional certification tracking
* Priority levels and validity periods
* Geographic and industry scope definition
* Integration with product categories

Configuration
=============

After installation, the module provides:

1. **Sequences**: Pre-configured sequences for procurement requests, categories, tags, and certifications
2. **Master Data**: Sample categories, tags, and certifications to get started
3. **Menu Structure**: Organized menu hierarchy under "Xportem" main menu

Usage
=====

Product Categories
------------------

1. Navigate to Xportem → Configuration → Master Data → Product Categories
2. Create hierarchical category structures for your products
3. Use the kanban view for visual organization
4. Set up parent/child relationships as needed

Product Tags
------------

1. Go to Xportem → Configuration → Master Data → Product Tags
2. Create tags with alternative names for better search
3. Associate tags with categories
4. Add keywords for enhanced matching capabilities

Example tag setup::

    Name: Stainless Steel
    Code: SS
    Alternative Names: acero inoxidable, inox, stainless
    Keywords: corrosion resistant, food grade, marine grade

Certifications
--------------

1. Access Xportem → Configuration → Master Data → Certifications
2. Define required certifications for your procurement
3. Set mandatory flags for critical certifications
4. Configure geographic and industry scope

Example certification::

    Name: ISO 9001
    Type: Quality Management
    Mandatory: Yes
    Priority: High
    Validity: 36 months
